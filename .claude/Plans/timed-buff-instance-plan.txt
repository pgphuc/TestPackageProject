1. BOI CANH VA VI SAO TACH KHOI TIMEDREGEN          (#1-boi-canh)
====================================================================

He TimedRegen (ATimedRegenInstance + Cooldown/DailyReset) model 1 LUONG
(count) hoi/reset theo thoi gian: GetCount/Add/Use/maxCount.

Timed buff khac BAN CHAT: khong co "count". No la 1 CONG (gate) boolean
+ 1 moc HET HAN. Vi du:
  - Infinite heart: trong N giay, UseHeart khong tru.
  - x2 coin / shield / boost: bat trong 1 cua so thoi gian.

API GetCount/Add/Use/MaxCount VO NGHIA voi buff. => phai la he RIENG,
khong nhet vao ATimedRegenInstance (giong ly do tach Cooldown vs DailyReset).

LUU Y QUAN TRONG: buff khong tu "chan" viec tieu resource khac. Vi du
infinite heart khong tu chan UseHeart — chinh cho TIEU heart phai hoi
buff truoc (IsActive?). Buff system chi tra loi "dang active hay khong"
+ "con bao lau". Logic "de len" nam o consumer/facade (ResourceManager).


====================================================================
2. MO HINH DU LIEU & API                             (#2-mo-hinh)
====================================================================

State toi thieu cho 1 buff "cua so thoi luong": 1 timestamp het han (UTC ticks).
  - IsActive  = DateTime.UtcNow.Ticks < expireUtcTicks
  - Activate(durationSeconds): STACK/EXTEND tu moc con lai
      base = IsActive ? expireUtcTicks : now;  expireUtcTicks = base + duration
      (giong logic SetInfiniteHeart hien tai trong ResourceManager)
  - GetRemainingSeconds(): max(0, expire - now) doi ra giay
  - Clear(): expireUtcTicks = 0 (tat ngay)

Lazy: chi tinh khi duoc hoi, khong can ticker moi frame -> dung ca khi
app dong/suspend (giong TimedRegen). UI tu poll GetRemainingSeconds cho
countdown, hoac nghe event OnTimedBuffChange khi Activate/Clear.

LUU Y: API o muc 6 dat o ABSTRACT base (xem 3.1). Cac field/logic tren la
cua variant dau tien (DurationBuff). Variant khac (charge-based...) co the
override hoac mo rong rieng.

3. QUYET DINH KIEN TRUC DA CHOT                     (#3-quyet-dinh)
====================================================================

3.1 [DA CHOT] LAM ABSTRACT INSTANCE.
  - `ATimedBuffInstance` (abstract, JBase) giu state/contract chung; subclass
    cho tung KIEU buff. Lam ngay de mo rong nhieu loai buff khac nhau ve sau
    (khong chi "cua so thoi luong").
  - Variant dau: `DurationBuffInstance` (cua so het han theo thoi luong).
  - Phan KIEU thoi gian dinh danh bang enum `BuffKind` (giong RegenKind):
      None=0, Duration=1 (them Charge=2... khi co variant moi).
  - Tach 2 truc giong he regen:
      BuffType  = DINH DANH resource buff (InfiniteHeart, X2Coin...) — int serialize.
      BuffKind  = KIEU thoi gian (Duration, ...) — quyet dinh subclass nao.
  - Manager + Config cung A-prefix abstract (convention + DI swap).

3.2 [DA CHOT] CO LAM CONFIG (TimedBuffConfig).
  - De sau con extend (default/max duration per-type, bat/tat buff qua designer).
  - Manager nap config luc init giong TimedRegenManager (lazy EnsureInitialized,
    factory theo BuffKind). KHONG auto-register an.
  - Config point toi thieu: type + kind + defaultDurationSeconds (+ maxDurationSeconds
    tuy chon, cap khi Activate). Field se chot lai luc viet.

3.3 Moc thoi gian: dung UTC xuyen suot (giong TimedRegen, chong lech clock).

3.4 Stack policy khi Activate luc dang active: CONG DON (extend) — giong
    SetInfiniteHeart hien tai. Neu muon "lay max" hay "ghi de" thi la
    quyet dinh khac, hoi user.

6.3 TimedBuffInstance.cs  (namespace JoyCraftSDK.Game)
    --- ATimedBuffInstance (abstract, JBase) — INVARIANT/CONTRACT chung ---
    Fields chung: BuffType type; SavedBuffPoint savedBuffPoint.
    Accessor protected: expireUtcTicks (get/set savedBuffPoint.expireUtcTicks).
    Public: BuffType Type => type.
    Ctor base: (BuffType type, SavedBuffPoint savedBuffPoint).
    RaiseChange() chung (giu o base):
      _eventBus.Raise(new OnTimedBuffChange {
          type = type, isActive = IsActive(),
          remainingSeconds = GetRemainingSeconds() });
    ABSTRACT (variant):
      bool  IsActive();
      void  Activate(float durationSeconds);
      float GetRemainingSeconds();
      void  Clear();
    (Neu thay nhieu logic dung chung giua cac variant tuong lai -> ha xuong
     base virtual. v1 cu de abstract, DurationBuff implement.)

    --- DurationBuffInstance : ATimedBuffInstance (variant "cua so thoi luong") ---
    Ctor: (BuffType type, SavedBuffPoint savedBuffPoint) -> base(type, savedBuffPoint).
    IsActive(): return DateTime.UtcNow.Ticks < expireUtcTicks;
    Activate(float durationSeconds):
      if (durationSeconds <= 0) return;
      long now = DateTime.UtcNow.Ticks;
      long addTicks = TimeSpan.FromSeconds(durationSeconds).Ticks;
      long baseTicks = IsActive() ? expireUtcTicks : now;
      expireUtcTicks = baseTicks + addTicks;
      RaiseChange();
    GetRemainingSeconds():
      long remain = expireUtcTicks - DateTime.UtcNow.Ticks;
      return remain <= 0 ? 0f : (float)TimeSpan.FromTicks(remain).TotalSeconds;
    Clear():
      if (expireUtcTicks == 0) return;
      expireUtcTicks = 0; RaiseChange();

6.4 TimedBuffManager.cs
    public class TimedBuffManager : ATimedBuffManager {
        private readonly Dictionary<BuffType, ATimedBuffInstance> instances = new();
        private bool isInitialized;
        private SavedBuffData savedBuffData => _playerProgressData.SavedBuffData;

        private void EnsureInitialized() {
            if (isInitialized) return;
            isInitialized = true;
            var timedBuffConfig = _gameConfig.timedBuffConfig;
            if (timedBuffConfig == null) return;
            foreach (var configPoint in timedBuffConfig.GetConfigPoints()) {
                RegisterBuff(configPoint);
            }
        }
        // Factory theo kind:
        public override void RegisterBuff(TimedBuffConfigPoint configPoint) {
            if (instances.ContainsKey(configPoint.type)) return;
            var savedBuffPoint = savedBuffData.GetOrCreatePoint(configPoint.type);
            ATimedBuffInstance instance = configPoint.kind switch {
                BuffKind.Duration => new DurationBuffInstance(configPoint.type, savedBuffPoint),
                _ => null
            };
            if (instance == null) { LogError($"BuffKind.None cho {configPoint.type}"); return; }
            instances[configPoint.type] = instance;
        }
        private ATimedBuffInstance GetInstance(BuffType type) {
            EnsureInitialized();
            if (instances.TryGetValue(type, out var instance)) return instance;
            LogError($"Buff {type} chua dang ky. Them vao TimedBuffConfig.");
            return null;
        }
        public override bool IsActive(BuffType type) {
            var instance = GetInstance(type); return instance != null && instance.IsActive();
        }
        public override void Activate(BuffType type, float durationSeconds)
            => GetInstance(type)?.Activate(durationSeconds);
        public override float GetRemainingSeconds(BuffType type) {
            var instance = GetInstance(type); return instance == null ? 0f : instance.GetRemainingSeconds();
        }
        public override void Clear(BuffType type) => GetInstance(type)?.Clear();
    }
    public abstract class ATimedBuffManager : JBase {
        public abstract void RegisterBuff(TimedBuffConfigPoint configPoint);
        public abstract bool IsActive(BuffType type);
        public abstract void Activate(BuffType type, float durationSeconds);
        public abstract float GetRemainingSeconds(BuffType type);
        public abstract void Clear(BuffType type);
    }
