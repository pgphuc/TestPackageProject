4. DIEM THEN CHOT PHAI QUYET (EVENT BRIDGE)         (#4-event)
==================================================================

VAN DE: UI nghe OnHeartChange. TimedRegen raise OnTimedRegenChange (khong co
HeartGainReason). Regen "ngam" xay ra BEN TRONG TimedRegen (luc GetCount/Use goi
ApplyPendingRegen) -> ResourceManager khong tu biet de raise OnHeartChange.

=> Phai BRIDGE OnTimedRegenChange(type==Heart) -> OnHeartChange. 2 cach, CHON 1:

CACH A (KHUYEN NGHI): ResourceManager SUBSCRIBE OnTimedRegenChange.
  - Khi type==Heart -> raise OnHeartChange { newAmount, changeAmount,
    success = true, gainReason = <reason dang cho> }.
  - Reason: giu 1 field tam `pendingHeartGainReason` (default TimePassed). Truoc khi
    goi AddHeart/Use set reason tuong ung; handler bridge doc roi reset ve TimePassed.
    -> regen ngam tu dong mang reason TimePassed; Add mang dung reason.
  - AddHeart/UseHeart (path thanh cong) KHONG raise OnHeartChange truc tiep — de bridge
    lam (tranh double-raise).
  - 2 path ResourceManager VAN tu raise (vi TimedRegen khong raise):
      * UseHeart khi infinite active (khong goi TimedRegen.Use) -> raise success=true.
      * UseHeart that bai (khong du) -> TimedRegen.Use tra false, khong raise -> RM raise success=false.
  Uu: 1 nguon su that cho moi thay doi count (ke ca regen ngam). Nhuoc: co field tam.

CACH B: KHONG subscribe. AddHeart/UseHeart tu raise OnHeartChange sau khi goi TimedRegen.
  - Nhuoc LON: regen ngam (TimePassed) xay ra trong GetHeartCount/Use se KHONG phat
    OnHeartChange -> UI khong update khi heart tu hoi. Phai goi GetSecondsUntilNext/poll.
  - Chi chap nhan neu UI heart KHONG dua vao event regen (chi poll moi giay). Kiem tra
    HeartView truoc khi chon. Don gian hon nhung de mat update.

=> KHUYEN NGHI CACH A. Hoi user xac nhan truoc khi code (anh huong UI behaviour).
   Kiem HeartView.cs xem no update theo event hay poll de chot.

7. CHI TIET: FACADE DELEGATION                      (#7-facade)
==================================================================

Trong ResourceManager (Heart region), thay ruot. RegenResourceType.Heart la "Heart".

  GetHeartCount()          -> return _timedRegenManager.GetCount(Heart);
  GetMaxHeart()            -> return _timedRegenManager.GetMaxCount(Heart);
  GetSecondsUntilNextHeart()-> if (IsInfiniteHeartActive()) return 0f;
                              return _timedRegenManager.GetSecondsUntilNext(Heart);
  ApplyPendingRegen()      -> _timedRegenManager.ApplyPendingRegen(Heart);
  AddHeart(amount, reason) -> pendingHeartGainReason = reason;          // CACH A
                              _timedRegenManager.Add(Heart, amount);
                              // KHONG raise o day — bridge lo (Cach A)
  UseHeart(amount)         -> ApplyPendingRegen() (qua GetCount/Use deu tu goi);
                              if (IsInfiniteHeartActive()) {
                                  raise OnHeartChange success=true, changeAmount=amount,
                                        newAmount = GetHeartCount(); return true;
                              }
                              bool ok = _timedRegenManager.Use(Heart, amount); // success -> bridge raise
                              if (!ok) raise OnHeartChange success=false, changeAmount=0,
                                             newAmount = GetHeartCount();
                              return ok;

  // INFINITE HEART giu nguyen (khong dung TimedRegen):
  SetInfiniteHeart / IsInfiniteHeartActive / GetInfiniteHeartRemainingSeconds: NHU CU.

BRIDGE (Cach A) — them handler + sub/unsub:
  private HeartGainReason pendingHeartGainReason = HeartGainReason.TimePassed;
  private void OnTimedRegenChangeHandler(ref OnTimedRegenChange e) {
      if (e.type != RegenResourceType.Heart) return;
      _eventBus.Raise(new OnHeartChange {
          gainReason = pendingHeartGainReason,
          success = true,
          newAmount = e.newAmount,
          changeAmount = e.changeAmount
      });
      pendingHeartGainReason = HeartGainReason.TimePassed; // reset ve default (regen ngam)
  }
  Subscribe(): _eventBus.SubscribeTo<OnTimedRegenChange>(OnTimedRegenChangeHandler);
  Unsubscribe(): _eventBus.UnsubscribeFrom<OnTimedRegenChange>(OnTimedRegenChangeHandler);

LUU Y: kiem dau "->" khi changeAmount<0 (Use) van map dung dau cho UI. TimedRegen.Use
raise changeAmount = -amount; OnHeartChange cu khi Use truoc day raise changeAmount=amount
(duong). Neu UI dua vao dau -> doi chieu HeartView de map cho khop (co the can Math.Abs
hoac giu dau). CHOT theo HeartView thuc te.
